WEEK 8:

What did you learn today?

Typography principles and how to make responsive websites using CSS grids.

What was new / interesting?

We looked at the impact of font on the overall feel of a website. Different sizes, weights, line-spacings and styles are all very important things to consider.

Learning how the use the "@media" query showed me how to change the grid layout depending on the monitor sizes to make my website look good on all devices.

How can it be applied in web design?

By considering all the above variables you can chose a font which will be readable, fit the tone of your website and produce an effective hierachy.

How does it influence you as a web-designer?

When designing I am going to highly consider my font so it alligns with the user group and the intended tone.
